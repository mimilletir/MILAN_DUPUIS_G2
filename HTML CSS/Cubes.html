Les Cubes et les Boutons

Les Cubes sont des éléments basiques de gameplay dans Portal.
Le bouton est utilisé dans les salles de test et est activé quand le joueur ou un Cube est placé dessus.

Les boutons sont présents dans pratiquement toutes les salles de test et provoquent des effets variés quand ils sont activés. Ils peuvent libérer un Cube ou ouvrir et fermer une porte d'accès menant vers la suite du parcours. C'est au joueur de trouver quand et comment utiliser le ou les boutons pour avancer. Le bouton se désactive quand il n'y a rien qui appuie dessus.
