Les Gels

Les Gels sont des liquides semblables à de la peinture qui impliquent différents effets quand ils sont appliqués sur des surfaces ou des objets.

Le gel répulsif est une substance de couleur bleu ciel qui repousse les objets, tout sujet de test ou objet qui frappera une surface couverte de gel répulsif rebondira. La hauteur du rebond sera proportionnelle à la hauteur depuis laquelle le joueur est tombé avant de toucher le gel. Les objets comme les cubes de stockage lestés et les Tourelles peuvent être recouvert de gel répulsif, les faisant rebondir partout de façon ératique.
Le gel propulsif est une substance orange qui réduit fortement la friction entre les objets et les surfaces sur lesquels ceux-ci se déplacent. Les objets couverts de gel propulsif ont aussi une friction réduite, les faisant glisser et bouger plus facilement.
Le gel conversif est un gel blanc fait à base de roche lunaire qui permet de rendre une surface capable de supporter le placement de portails, même si elle n'en est pas de base capable.
