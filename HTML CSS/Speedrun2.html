Speedrun

Speedrun de Portal 2

Le speedrun est une pratique liée aux jeux vidéo dans laquelle le but est d'atteindre le plus rapidement possible un objectif donné, le plus souvent terminer le jeu.
Il existe plusieurs catégories différentes pour un même jeu.

Les différentes catégories de Speedrun pour Portal 2 sont :
- any% : finir le jeu le plus rapidement possible (la vidéo ci-contre correspond au WR de cette catégorie)
- any% Glitchless : finir le jeu le plus rapidement possible sans utiliser de bugs
- Least Portals : finir le jeu en utilisant le moins de portails
- Co-op any% : Finir le mode Co-op le plus rapidement possible
- Solo Co-op : Finir le mode Co-op le plus rapidement possible en jouant seul
